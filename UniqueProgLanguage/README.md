A programming language based on the one liners of Arnold Schwarzenegger.

```

TALK TO THE HAND "Hello Local Hack Day Share !"

YOU HAVE BEEN TERMINATED

```
#### To run the above program 
- `java -jar ArnoldC.jar LHDShare.arnoldc`
- `java LHDShare`
